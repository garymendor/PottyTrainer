# Potty Trainer

A plugin that simulates your character having bodily functions, intended for roleplay involving ABDL, desperation, and omorashi.

Periodically, messages will appear in your chat windown to indicate your increasing need to urinate or defecate. If you ignore
those notifications long enough, your body will take matters into its own hands! These messages can be seen only by you;
it is up to you how to incorporate them into a bathroom-focused roleplay.

## Future features

On the roadmap:
* Underwear! Determine what kind of undies you're wearing (if any); if you're still wearing them when you let go,
  you will wet or soil those undies!
* Moodles integration.
* Posting potty chat messages to other than yourself.
* Simpler character configuration.
